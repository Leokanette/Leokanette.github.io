FUELLY : A Booking, Delivery, and Navigating App

This mobile application does not just give you the best choice to fuel up your engine, this online navigating app also does allow delivery service to certain people who dried up their fuel tank during the long ride. Fuelly is an online navigating map that allows you to search for the nearest gasoline station using an online generated map and can also use the navigating option. their price rates as well as to call for gasoline delivery if ever you run out of gas somewhere. It also shows the the designated prices for each types of engine fuels for each designated gasoline station that the target user might want to view as well the availability time and closing time of a certain gasoline station.

Fuelly is one of my projects which involves on building user interfaces using figma. The project aimed to create a wireframe and a High Fidelity Prototype of a certain application that we need to enhance.
